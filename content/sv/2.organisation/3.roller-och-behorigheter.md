# Roller och behörigheter

Varje medlem i organisationen har en **roll**. Rollen avgör vad personen får se och göra. Du tilldelar roller på sidan [Medlemmar](/sv/organisation/medlemmar).

## Rollerna i korthet

| Roll | Vad rollen normalt får göra |
| --- | --- |
| **Ägare** | Full kontroll över organisationen, inklusive medlemmar, inställningar och fakturering. |
| **Administratör** | Hanterar organisationen och dess medlemmar, men har inte ägarens fakturerings­ansvar. |
| **Operatör** | Arbetar i organisationen och kan se medlemslistan, men hanterar inte medlemmar eller inställningar. |
| **Medlem** | Standardroll med åtkomst till organisationens innehåll. |
| **Visare** | Läsåtkomst — kan se men inte ändra. |
| **Support** | Insynsroll för supportändamål, inklusive att läsa granskningsloggar. |

::note
Exakt vad varje roll kan göra beror också på vilka moduler och funktioner som är aktiverade i just din organisation. Tabellen beskriver det vanliga utfallet.
::

## Så här tilldelar du en roll

1. Gå till [Medlemmar](/sv/organisation/medlemmar).
2. Hitta personen i listan.
3. Välj rollen i rullgardinsmenyn i kolumnen **Roll**.

Ändringen sparas direkt.

::tip
Ge varje person den lägsta roll som räcker för deras arbete. Det är enklare att höja behörigheten senare än att reda ut för vida rättigheter.
::

## Anpassade behörigheter per modul

Utöver grundrollen kan du finjustera vad en medlem får göra i enskilda moduler. Det gör du via knappen **Behörigheter** på medlemmens rad, som öppnar en panel där modulerna listas.

För varje behörighet kan du välja:

- **Tillåt** — ge medlemmen den specifika behörigheten.
- **Neka** — blockera behörigheten även om rollen normalt skulle ge den.
- **Ärv** — följ det som rollen ger som standard (ingen särskild inställning).

Du kan också snabbt sätta **Tillåt alla**, **Neka alla** eller **Återställ (ärv)** för en hel modul.

::note
Vissa behörigheter kan vara markerade som *Blockerad av policy*. Det betyder att organisationens inställningar hindrar behörigheten, och då går den inte att tillåta för enskilda medlemmar.
::

När du gjort ändringar markeras de som **Osparade**. Klicka på **Spara** för respektive modul, eller **Spara alla** för att spara allt på en gång. Medlemmar som har anpassade behörigheter får en markering i medlemslistan och kan filtreras fram där.

::warning
Anpassade behörigheter lägger sig ovanpå rollen och kan vara lätta att glömma. Använd dem sparsamt och se över dem regelbundet, så att åtkomsten håller sig förutsägbar.
::
