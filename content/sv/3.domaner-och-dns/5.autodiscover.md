# Autodiscover

Autodiscover hjälper dig att hitta DNS-zoner som hör till din organisation och aktivera dem i portalen, så att du kan börja hantera deras poster. Det är ett smidigt sätt att komma igång om du har flera domäner.

## Kör autodiscover

1. Gå till **Domäner & DNS** och öppna **Autodiscover** (du når den också via DNS-zoner).
2. Sökningen startar automatiskt när du öppnar sidan. Du kan också klicka **Kör Autodiscover** för att söka igen.
3. När sökningen är klar listas de zoner som hittats. Varje zon visar domännamn och vilken server den ligger på.

## Aktivera zoner

För varje hittad zon ser du en status:

- **Aktiverad** — zonen tillhör redan din organisation och hanteras i portalen.
- **Tillgänglig** — zonen kan aktiveras.

Så här aktiverar du zoner:

1. Markera de zoner du vill aktivera med kryssrutorna (tillgängliga zoner är förvalda).
2. Klicka **Aktivera valda** för att aktivera de markerade, eller **Aktivera alla** för att aktivera samtliga tillgängliga.
3. Listan uppdateras och de aktiverade zonerna visas nu som **Aktiverad**.

När en zon är aktiverad dyker den upp under [DNS-zoner](/sv/domaner-och-dns/dns-zoner) där du kan öppna den och hantera dess poster.

## Om inga zoner hittas

Om sökningen inte hittar några zoner visas ett meddelande om det. Det betyder oftast att din domän ännu inte är förberedd för automatisk identifiering.

::note
För att en domän ska kunna hittas automatiskt behöver den vara konfigurerad så att den kan kopplas till din organisation. Hittar autodiscover inte dina domäner, kontakta din administratör så hjälper de dig att förbereda eller aktivera dem.
::
