# Branding

You can customize the portal's appearance to match your brand under **Settings → Branding**.

::note
Partners set branding for a customer under **Partner → Tenants → [customer] → Branding**. Tenant level takes precedence over distributor level, which takes precedence over the system default.
::

## What you can customize

- **Logo** — upload your own logo (JPG, PNG, SVG or WebP, max 2 MB). Shown in the navigation. You can change or remove it at any time.
- **Accent color** — color for buttons, links and highlights. Pick it with the color picker or enter a hex code (e.g. `#005BFF`). There are preset color suggestions, and a **Reset** button.
- **Navigation color** — background color for the menu/header. Same choices: color picker, hex code, presets and reset.

Click **Save** for each section.

## Branding on the sign-in page

Branding on the sign-in page itself activates automatically once you have a [custom domain](/en/organization/custom-domain) that is **Active**. The sign-in page then matches your logo and colors for that domain.
